Overview
========

`HITMAN`_, Hermite Interpolation of Trajectories (HIT) and Measurement synthesis for Analysis of Navigators (MAN), is
a python library for interpolating flight trajectories and generating synthetic IMU
measurements.

As an introduction check out the
:doc:`notebooks/how_to_compensate_coning`. Additional examples provided
as how-to guides.
