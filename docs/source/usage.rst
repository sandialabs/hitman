Oragnization
============
Primary methods for generating integrated-sensor outputs are provided in
:mod:`hitman.gyro`. For example, :func:`hitman.gyro.functional_factory`.

Exponential coordinates are used when representing group elements of :math:`SO(3)`
through the class :class:`hitman.rotation.LieExponential`.

Notation
========
A brief note on vairable names. Subscripts precede superscripts. For example,
:math:`x_a^b` should receive a variable name `x_a_b`.

Preference is to local-to-global rotations.

We take the contentious stand that names of matrix-valued variables may begin with capital letters.
